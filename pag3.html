Elementos de un Sistema de Informacion computacional

EQUIPO COMPUTACIONAL:

hardware necesario para que el sistema de informacion funcione,integrado por computadoras y equipo periferico conectado a ellas.

RECURSOS HUMANOS:

son las personas que interactuan co el sistema de informacion, suministran datos o utlilizan el sistema para generar datos.

FUENTE:

son los datos e informacion que se introducen en el sistema, es decir,son las entradas requeridas para que el sistema informatico funcione.

PROGRAMAS.

sotfware ejecutado por la computadora, el cual procesa la informacion de entrada para generar los resultados esperados.

TELECOMUNICACIONES

abarca el software y hardware que permite la informacion en forma electronica de texto e imagenes, entre otros.

POLITICAS Y REGLAS DE OPERACION:

aplicadas a los procedimientos y mecanismos para hacer que una aplicacion de computadora trabaje de acuerdo con estas o regir la paerte funcional del proceso admistrativo de la empresa.
